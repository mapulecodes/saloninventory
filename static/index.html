Home

HTTP Methods Path Description
GET / Home page

Users

HTTP Methods Path Description
GET /users Retrieve all users
GET /users/:id Retrieve a specific user by ID
POST /users/register Register a new user
PATCH/PUT /users/update/:id Update user information by ID
DELETE /users/delete/:id Delete a user by ID

Products/Items

HTTP Methods Path Description
GET /products Retrieve all products
GET /products/:id Retrieve a specific product by ID
POST /products/addProduct Add a new product
PUT/PATCH /products/update/:id Update product information by ID
